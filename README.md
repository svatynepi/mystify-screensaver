# Mystify Screen Saver for macOS

![Preview](preview.png)

A Swift recreation of the classic 90s Windows Mystify screen saver, built as a native macOS `.saver` screen saver.

## Build

```bash
./scripts/build-saver.sh
```

Output: `build/Mystify.saver`

## Install locally

Double-click `Mystify.saver` to install it into **System Settings → Screen Saver**.
